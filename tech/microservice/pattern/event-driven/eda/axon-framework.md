# ✔️ Axon Framework

Apache 2 .0

* 상업적 사용 가능
* 소스 코드 공개 필요 X
* 저작권 고지 포함 필요 O



Language :  Java Framework

Feature :

* DDD
* CQRS ( Command Query Responsibility Segregation )
* ( Axon Framework + Axon Server ) 구성



Axon Server&#x20;

* Coordinator (조율자) 역할
  * Command Bus
  * Event Bus
  * Event Store
* 발행한  Event  저장
* Application 상태 관리
* Queue 유량 조절



Concept

1. Aggregate (집합)
2. Command (명령)
3. Event (이벤트)
4. Command Handler (핸들러)
5. Event Handler (핸들러)
